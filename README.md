# Cellular Automata Simulations (Python)

This repository contains selected Python projects developed as part of the **Discrete Modelling** coursework at AGH University of Krakow.

The projects focus on **cellular automata (CA)**-based simulations and related computational modelling techniques used to represent dynamic systems in 1D and 2D spaces. The coursework introduced concepts such as neighborhood rules, periodic boundary conditions, grid-based simulations, and selected elements of stochastic modelling.

## Project Scope

The repository includes implementations and experiments related to:

- **1D and 2D cellular automata**
- **Periodic boundary conditions**
- **Neighborhood-based state transition rules**
- **Grid-based simulation models**
- **Selected microstructure / growth process simulations**
- **Introductory Monte Carlo-based modelling concepts**
- **Visualization of simulation results in Python**

## Technologies Used

- **Python**
- **NumPy**
- **Matplotlib**
- Jupyter Notebook / Python scripts (depending on project version)

## Learning Focus

Through these projects, I developed practical skills in:

- implementing rule-based simulation algorithms,
- working with multidimensional arrays and grid representations,
- modelling dynamic systems using computational methods,
- visualizing simulation behavior and results,
- translating theoretical modelling concepts into working Python code.
